# Todo with Express Mongodb and Docker
A simple Todo app 

## Features
- Todo Create
- Todo Update
- Todo Filter
- Todo Delete
- Todo View By Id

---

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/surujmiah6896/todo-with-express-mongodb.git
cd todo-with-express-mongodb
```


## Install Dependencies

```bash
docker compose up
```
